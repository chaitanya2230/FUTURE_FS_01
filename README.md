# Madimi Chaitanya Sai - Portfolio

A simple, modern, and lightweight personal portfolio website built with pure HTML, CSS, and JavaScript.

## 🚀 Technologies Used
*   **HTML5** for semantic structure
*   **Tailwind CSS** (via CDN) for rapid styling and responsive design
*   **Vanilla JavaScript** for DOM manipulation, smooth scrolling, scroll-spy, and animations
*   **Lucide Icons** & **DevIcons** for iconography

## ✨ Features
*   **Zero Dependencies:** Completely dependency-free and lightning fast. No `node_modules`, no build step.
*   **Glassmorphism UI:** Modern dark theme with beautiful subtle glow and blur effects.
*   **Fully Responsive:** Looks perfect on mobile, tablet, and desktop devices.
*   **Scroll Animations:** Elements fade and slide into view seamlessly using Intersection Observers.
*   **Dynamic Typing Effect:** Custom built hero section typing animation.

## 💻 Sections Included
1.  **Home:** Intro and dynamic code terminal card.
2.  **About Me:** Quick introduction and background.
3.  **Skills:** Categorized skills (Languages, Web, Databases, Core CS, AI/ML).
4.  **Projects:** Featured projects with links.
5.  **Competitive:** LeetCode, CodeChef, and HackerRank statistics.
6.  **Certifications:** Verified certifications from various platforms.
7.  **Contact:** Social links and an interactive contact form interface.

## 🛠️ How to run locally
Because this project uses vanilla web technologies, you do not need to install anything. 
Simply open `index.html` in any web browser of your choice.

## 🌐 Deploying
This project can be instantly deployed to any static hosting provider like GitHub Pages, Vercel, or Netlify by simply uploading the files.
